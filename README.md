# Development Blog for `Six Days at DHBW`

Here you can find the files that build the development blog for the game `Six Days at DHBW`.
You can read the blog on https://6semesteratdhbw.github.io/dev-blog/welcome.html, where it is automatically deployed on push to the main branch.

It is built using [Writerside](https://www.jetbrains.com/help/writerside/getting-started.html).
